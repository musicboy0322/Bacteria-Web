# 細菌查詢網站
* 解決問題：因為同學做完實驗後有大量的實驗結果，並且還自己分類，原本使用 Excel 記錄，但後來發現這樣太不直覺，後續當資料多時很難去使用，因此與他合作開發一個細菌查詢網站，使其作業上更方便。

* 使用工具：
  * 網頁製作：HTML、CSS、JavaScript
  * 後端框架：Node.js
  * 資料庫：MongoDB
  
* 使用技術：Express + EJS + Mongoose

* 目前功能：
  * 顯示目前擁有的細菌資料
  * 新增細菌資料。
  * 刪除細菌資料。
